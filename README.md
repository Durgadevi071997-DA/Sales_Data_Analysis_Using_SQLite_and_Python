
# SQLite Database using Python in Sales Data


## Project Overview

This project demonstrates how to use SQL queries inside Python to analyze sales data stored in a SQLite database. The project involves creating a small SQLite database, retrieving sales information using SQL queries, loading the results into Python using Pandas, and visualizing the output with a bar chart using Matplotlib.

## Business Objectives

The main objective of this project is to:

Connect Python to a SQLite database.

Write and execute SQL queries.

Calculate total quantity sold for each product.

Calculate total revenue generated by each product.

Load SQL query results into a Pandas DataFrame.

Visualize sales revenue using a bar chart.

## Tools and Technologies Used

Python

SQLite

sqlite3 Library

Pandas

Matplotlib

Jupyter Notebook / VS Code
## Findings

Revenue was successfully calculated for each product.

Total quantity sold was summarized product-wise.

Product performance was visualized using a bar chart.

SQL and Python were successfully integrated for data analysis.
## Outcome of Project

By completing this project, the following skills were developed:

Understanding SQLite databases

Writing basic SQL queries

Using aggregate functions such as SUM()

Applying GROUP BY for data summarization

Importing SQL data into Python

Working with Pandas DataFrames

Creating visualizations using Matplotlib

Performing basic sales analysis
## Conclusion

This project successfully demonstrated how SQL and Python can be integrated to perform sales data analysis. Using SQLite, Pandas, and Matplotlib, meaningful insights were generated from sales data, and the results were effectively visualized through a bar chart.
